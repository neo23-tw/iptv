## ▶️ 使用方法

1. **下載 m3u 文件**  
   - 點擊所需 m3u 文件名，進入文件頁面後點擊右上角的 `Raw` 按鈕，然後右鍵選擇“另存為”下載到本地。

2. **導入 IPTV 播放器**  
   - 推薦播放器：
     - [😈 強烈推薦Charming Player  PC端播放器](https://github.com/CharmingCheung/CharmingTVBox/releases)

   - 打開 VLC
   - 依次點擊 `媒體` → `打開文件...`，選擇下載的 m3u 文件
   - 或 `媒體` → `打開網絡串流`，粘貼 m3u 文件中的鏈接地址
